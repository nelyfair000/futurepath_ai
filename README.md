# 🤖 AI职业导航：性格测试 + 市场分析 + 学习计划

> 用AI帮你找到最适合的职业方向，告别迷茫


## 📖 项目简介

面对五花八门的职业选择，你是否也感到迷茫？“AI职业导航”通过**科学的性格测试 + 实时市场数据分析 + 个性化学习路径规划**，帮你找到最适合自己的职业方向，并告诉你**怎么入门、怎么成长**。


## ✨ 核心功能

- 🧠 **AI性格分析**：基于MBTI等模型，深度解析你的性格特质
- 📊 **实时市场趋势**：通过MCP连接招聘数据，分析职业需求、薪资范围、增长预期
- 🎯 **智能职业推荐**：结合性格+市场，推荐最匹配的3个职业及匹配度
- 📚 **专属学习计划**：按阶段生成学习路径，推荐具体课程和项目

## 🚀 快速开始

### 前置要求
- Node.js 18+
- npm 或 yarn
- OpenCode CLI (`npm install -g opencode-ai`)

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/futurepath_ai.git
cd ai-career-navigator

# 2. 安装前端依赖
npm install

# 3. 启动前端开发服务器
npm run dev

# 4. 新开终端，启动OpenCode后端服务
opencode serve --port 3001
